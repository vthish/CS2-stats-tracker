# 🎮 CS2 Stats Tracker

A dynamic, fast, and responsive Counter-Strike 2 player statistics tracker built with React, Vite, and the Steam Web API.

🔗 **Live Demo:** [https://admin-cs2-tracker.netlify.app/](https://admin-cs2-tracker.netlify.app/)

## ✨ Features
* 🔍 Search players by SteamID64 or Custom Profile Name (Vanity URL).
* 📊 Displays real-time combat statistics (Kills, Deaths, Headshots, etc.).
* 🎨 Modern Glassmorphism UI with smooth hover animations.
* ⚡ Fully responsive design for mobile and desktop.

## 🛠️ Technologies Used
* React + Vite
* Axios (API Requests)
* CSS3 (Glassmorphism & Animations)
* Steam Web API

## 🚀 How to Run Locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file and add your Steam API Key: `VITE_STEAM_API_KEY=your_api_key_here`
4. Run `npm run dev` to start the local server.
